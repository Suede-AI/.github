As AI turns music, media, identity, and creative work into programmable, remixable assets, the hard problem is no longer making content — it is proving who made it, what rights attach to it, and how value should flow when machines start using it. Suede Labs AI is the ownership layer that answers those questions in software: prove the work, register the right, program the license, route the royalty, and expose the result to paid agents.

This is infrastructure, not a single app. AI music generation is the first wedge and a core media use case, but the same proof, registry, licensing, and royalty primitives apply to vocals, sound, images, video, likeness, training data, and generative outputs.

## What we ship

| Capability | Description |
|---|---|
| **Programmable IP** | Register, verify, and route rights for creator-owned media |
| **Provenance** | Verifiable authorship and a checkable record of creation for AI-native assets |
| **Licensing** | Machine-readable sync, sample, master, and likeness rights agents can read and respect |
| **Royalty routing** | Creator-defined splits and cross-chain settlement across the surfaces where work appears |
| **Agent commerce** | Pay-per-call x402 endpoints and ACP-ready surfaces for AI agents to discover and transact |
| **Multichain rails** | Credits, Stripe, Base, Solana, and x402 settlement |
