As AI makes media easier to generate, remix, license, and distribute, creators need a stronger ownership layer underneath all of it. Suede makes IP easier to **register, verify, license, route, and monetize** — across products, multiple chains, and agent-driven commerce.

Music is the first wedge. The thesis covers any creator-owned IP: songs, video, likeness, training data, generative outputs.

## What we ship

| | |
|---|---|
| **Programmable IP** | Register, verify, and route rights for creator-owned media |
| **Provenance** | Verifiable authorship for AI-native creative assets |
| **Licensing** | Sync, sample, master, and likeness workflows on-chain |
| **Royalty infrastructure** | On-chain splits and cross-chain settlement |
| **Agent commerce** | x402 paid endpoints and ACP-ready surfaces for AI agents |
| **Multichain rails** | Base, Solana, Ethereum, Avalanche, omnichain |
